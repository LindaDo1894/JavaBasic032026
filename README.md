# JavaBasic032026 📚

A comprehensive Java fundamentals learning resource covering essential programming concepts with practical examples and hands-on exercises.

## 📖 Overview

This repository is designed for beginners learning Java programming. It provides a structured approach to mastering core Java concepts through organized modules, each focusing on a specific topic with clear examples and implementations.


## 🎯 Topics Covered

### 1. **Variables** (`src/Variables/`)
Learn how to declare and use variables in Java:
- Variable declaration syntax
- Variable naming conventions
- Variable assignment and reassignment

**Example:**
```java
String name = "Thi Linh Do";
int age = 31;
String address = "Blacktown, Sydney";
```

### 2. **Data Types** (`src/DataType/`)
Understand Java's primitive and reference data types:
- **Integer types:** byte, short, int, long
- **Floating-point types:** float, double
- **Boolean type:** true/false
- **Reference types:** String

**Example:**
```java
byte number1 = 0;
int number3 = 21000000;
double numberPi = 3.14444D;
boolean citizenship = true;
```

### 3. **Operators** (`src/Operators/`)
Master arithmetic, logical, and comparison operators

### 4. **Conditional Statements** (`src/ConditionIFELSE/`)
Control program flow with if, else if, and else statements

### 5. **Switch Case** (`src/SwitchCase/`)
Implement multi-way branching with switch-case statements

### 6. **For Loops** (`src/ForLoop/`)
Iterate through code blocks with various loop constructs

### 7. **Arrays** (`src/Array/`)
Work with arrays for storing multiple values

### 8. **String Handling** (`src/HandleString/`)
Manipulate and process strings effectively

### 9. **Data Collections** (`src/DataCollections/`)
Explore List, Set, and Map collections

### 10. **Methods** (`src/Method/`)
Define and call methods with different parameters and return types

### 11. **Static Members** (`src/StaticJava/`)
Understand static variables and static methods

## 🚀 Getting Started

### Prerequisites
- Java Development Kit (JDK) 8 or higher
- Any Java IDE (IntelliJ IDEA, Eclipse, VS Code with Java extensions)

### Clone the Repository
```bash
git clone https://github.com/LindaDo1894/JavaBasic032026.git
cd JavaBasic032026
```

### Open in IDE
1. Open your preferred IDE
2. Select "Open Project" and navigate to the cloned directory
3. Select the project folder

### Run Examples
Navigate to any module (e.g., `src/Variables/DemoVariable1.java`) and run the main method:
```bash
javac src/Variables/DemoVariable1.java
java Variables.DemoVariable1
```

## 📚 Learning Path

**Recommended order to learn:**

1. ✅ Variables - Start with variable basics
2. ✅ Data Types - Understand available data types
3. ✅ Operators - Learn to perform operations
4. ✅ Conditional Statements - Control flow with conditions
5. ✅ Switch Case - Alternative to if-else
6. ✅ For Loops - Repetition and iteration
7. ✅ Arrays - Store multiple values
8. ✅ String Handling - Text processing
9. ✅ Methods - Code reusability
10. ✅ Data Collections - Advanced data structures
11. ✅ Static Members - Class-level variables and methods
